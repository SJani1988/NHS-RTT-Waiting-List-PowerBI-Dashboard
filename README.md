
# NHS RTT Waiting List Performance Dashboard – April 2026

## About This Project

I created this Power BI dashboard to analyse NHS Referral to Treatment (RTT) waiting list performance.

The aim of this project was to transform healthcare activity data into a clear and interactive dashboard to understand completed RTT pathways, including **admitted and non-admitted pathways**, and explore variation across clinical specialities and NHS providers.

This project combines my healthcare background with my interest in data analytics, using Power BI to create meaningful insights that can support healthcare performance monitoring and service improvement.

---

## Project Aim

The dashboard was developed to answer key questions:

- How many RTT pathways were completed during the reporting period?
- How do admitted and non-admitted pathways compare?
- Which specialities have the highest RTT activity?
- Which providers manage the highest pathway volumes?
- Where are longer waiting times observed?

## Key Insights

Analysis of the April 2026 RTT reporting period identified several important patterns:

- **1,121,074 completed RTT pathways** were analysed for the reporting period.

- The dataset included **279 NHS providers and 23 clinical specialities**, providing a comprehensive overview of RTT performance.

- **Other – Medical Specialties** recorded the highest number of completed RTT pathways, followed by **Ophthalmology** and **Trauma & Orthopaedics**.

- **Completed Non-Admitted Pathways accounted for a larger share of RTT activity compared with Completed Admitted Pathways.**

- The average RTT volume per provider was **339.62 pathways**, although activity varied considerably between providers.

- **Manchester University NHS Foundation Trust** reported the highest RTT pathway volume, followed by **Mid and South Essex NHS Foundation Trust** and **University Hospitals of Leicester NHS Trust**.

- Average RTT waiting times were highest in **Other – Surgical Services, Other – Medical Services, and Dermatology**, indicating comparatively longer treatment pathways in these specialities.

- RTT pathway volumes were concentrated among a smaller number of NHS providers, while many providers managed lower patient volumes.

- The dashboard highlights variation in RTT performance across specialities and providers, supporting targeted operational planning and resource allocation.
